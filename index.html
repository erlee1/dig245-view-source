DIG 245 View Source

How do you view the source code of a webpage on Google Chrome?

  1. Open Google Chrome.
  2. Open any webpage.
  3. Right-click anywhere on the webpage.
  4. Select View page source at the very bottom of the menu.
    • Alternatively, press Ctrl+U.
  5. You've arrived!

What else can you do with Dev Tools?

  • Run JavaScript code in the console.
  • Change the text in a preview mode.
  • Delete HTTP cookies.

What is using DevTools like?

Using DevTools is like taking an X-ray: you get to see the bones of the webpage and maybe diagnose any problems you might not have noticed just from looking at the surface.
